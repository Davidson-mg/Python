
Para instalar precisa estar como venv: source venv/bin/activate
pip install django-crispy-forms

(dica: vc pode usar o comando 'pip list' para verificar se a aplicação realmente foi instalada)


vai no arquivo settings do seu projeto e adicione "'crispy_forms'," em INSTALLED_APPS

Exemplo:
INSTALLED_APPS = [
...
...
'crispy_forms',
]



Ainda no arquivo settigns, adicione também está variavel, se atentando a versão do seu bootstrap
CRISPY_TEMPLATE_PACK = 'bootstrap4'


Vai no html que vc quer inserir e adicione no topo esta tag
{% load crispy_forms_tags %}


Pode ser que dê erro ao abrir a pagina do form. Caso isso aconteça, reinicie o nginx e o gunicorn. 
sudo systemctl restart nginx
sudo systemctl restart gunicorn



Exemplo aplicando no form:

            <form method="POST" action="#envia_comentario">
                {% csrf_token %}

                <!-- {{form}}--> 

            
                {% for hidden_field in form.hidden_fields %} <!-- Esse hidden_fileds será uma lista vazia e é
                     uma variavel do form mesmo-->
                {{ hidden_field }}
                {%endfor%}

                {% for visible_field in form.visible_fields %} <!-- Esse hidden_fileds será uma lista vazia -->


                    {{ visible_field|as_crispy_field }} 

                {%endfor%}

                {% include 'parciais/_messages.html' %}
                <input id="envia_comentario" type="submit" class="btn btn-primary" value="Enviar">

            </form>
